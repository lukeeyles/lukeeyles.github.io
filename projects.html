My Projects

These are some of the projects I've worked on in my engineering degree.

Tic-Tac-Toe Bot


For my major project in the univeristy subject Robotics, I worked in a team with two others to program a Dobot Magician to play Tic Tac Toe using Matlab with Peter Corke’s Robotics Toolbox. We used a webcam to detect when the player had finished their turn and where they had placed their tile. Then, the robot would use a custom inverse kinematics function to pick up and place its own tile, trying to beat the player. We also created a GUI to enable easy operation of the robot, including starting and stopping the game, jogging the end effector of the robot in cartesian coordinates, or moving individual joints.

Wall Building Robots


As part of the university subject Robotics, I programmed a simulated UR3 and UR5 robot to work together to build a wall in Matlab, using Peter Corke’s Robotics Toolbox. I used inverse kinematics to find waypoints for the robots to pick up and place the bricks, and quintic polynomial trajectory planning to smoothly interpolate between these waypoints. To increase efficiency, the robots picked up the bricks at the same time, and then took turns placing the bricks to avoid collisions.

Simulated Aircraft Interceptor

In the university subject Programming for Mechatronic Systems, one project was to use C++ with ROS to predict the trajectory of enemy aircraft and intercept them in a simulated airspace. This involved using threads to process incoming radar messages from ROS to determine where the closest enemy is. The program then predicted the time to intercept and extrapolated the enemies trajectory by that time, then generated control instructions in the form of a twist ROS message for the friendly aircraft to intercept.

TurtleBot Finding Cereal


For the major project in the subject Sensors and Control, I worked in a team with two others to program a TurtleBot to scan for cereal boxes, recognise the chosen box, and go the box by driving along its normal vector. My leadership skills were demonstrated in this project, as I worked early in the project to draw up an outline of the code structure and divide the work among the team. This ensured the success of the project as everyone was on the same page early in the project, and everyone had a specific part of the project that they were responsible for.

Matlab was used with ROS to program the system, so that the image processing toolbox and built-in matrix operations could be used. An RGBD camera mounted on top of the TurtleBot was used with the SURF feature detection algorithm to extract features from the images. These features were then compared against reference images using RANSAC, and if the right cereal box was in the image, it would be found. The marker’s location in the image was then transformed to a 3D location using the pinhole camera model. Then, the robot drives the shortest distance to the normal of the cereal box and turns to drive towards the box.

Arduino Security System

I programmed an Arduino security system that used a distance sensor mounted to a stepper motor to scan the room and detect any changes, as part of the university subject Mechatronics 2. An LCD shield was used to operate the system. The features of the system include password protection to access the menus, an environment calibration mode to set the reference data for the room, a security mode to detect any changes in the room which could operate on a timer, and a mapping mode which would convert the distance information to a 2D map of the room and print it out over serial.

Warman Competition 2019

As part of the Mechanical Design 1 subject, I worked in a team of 5 to create a robot to participate in the 2019 Warman Design and Build Competition. The goal of the challenge was to design a robot to lift plastic balls from one container and transport them to another container 2 metres away. To lift the balls, the method my team developed was to roll a large adhesive wheel across them, so that the balls would stick to the wheel. The shaft of the wheel was mounted to a pin joint so that it could be lifted in and out of the container, and a differential drive system was used to drive the robot so that its motions could be controlled predictably.
